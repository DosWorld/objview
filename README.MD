# OmfView

This is MS-DOS tools for investigation object formats:

* omfview - OBJ/LIB files in OMF format.
* rdfview - OBJ files in RDOFF2 format (NASM version 0.98.39).

# Using

        omfview objfile.obj
        rdfview rdffile.obj

or

        omfview "Long file name.obj"
        rdfview "Long file name.obj"
